# 🌾 CropYieldPrediction – Smart Agriculture AI System

An AI-powered agriculture platform that helps farmers predict crop yield, monitor weather, analyze satellite NDVI data, and receive smart crop & fertilizer recommendations.

---

## 🚀 Project Overview

CropYieldPrediction is a machine learning-based web application designed to improve agricultural productivity using:

* 📊 Predictive Analytics
* 🌦️ Real-time Weather Data
* 🛰️ Satellite NDVI Monitoring
* 🌱 Smart Crop & Fertilizer Suggestions
* 📱 Mobile-Friendly Weather Reports

---

## 🎯 Key Features

### 🏠 Dashboard

* Overview of farm statistics
* Recent predictions
* Weather summary
* Crop performance insights

### 🌾 Predict Yield

* Input soil data, rainfall, temperature, and crop type
* AI-based yield prediction
* Accuracy-focused ML model

### 📜 History

* View past predictions
* Track seasonal performance
* Export data reports

### 🌦️ Weather Module

* Live weather updates
* Temperature, humidity, rainfall
* Mobile-supported weather report

### 🌱 Recommendations

#### 🌿 Crop Recommendations

* Suggests best crops based on soil & weather
* Seasonal suitability analysis

#### 🧪 Fertilizer Guide

* Recommends fertilizers based on soil nutrients (N, P, K)
* Balanced nutrient planning

### 🛰️ Satellite NDVI Data

* Normalized Difference Vegetation Index
* Crop health monitoring
* Early stress detection
* Growth stage visualization

### 📅 Crop Calendar

* Seasonal crop planning
* Sowing & harvesting timelines
* Region-based recommendations

### 🤖 Smart Prediction System

* Combines weather + soil + historical yield
* AI-driven decision support
* Improved farming efficiency

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript / React

* **Machine Learning:** Scikit-learn / TensorFlow
* **Database:** MySQL / MongoDB
* **Weather API:** OpenWeather API
* **Satellite Data:** NDVI Remote Sensing Data
